# Simple PHP MVC Website

Учебный проект сайта-визитки, реализованный на кастомной MVC-архитектуре. Проект демонстрирует принципы разделения бизнес-логики, отображения и управления запросами.

## 🚀 Функционал
* **MVC Core:** Собственное ядро системы (Model-View-Controller).
* **Routing:** Чистые URL (Friendly URLs) реализованные через `.htaccess` и класс `Route`.
* **Статические страницы:** "Главная", "О нас", "Контакты".
* **Централизованный шаблон:** Использование единого `template_view.php` для всех страниц.

## 🛠 Технологии
* PHP 8.x
* Apache (модуль mod_rewrite)
* Пакет PHP-GD (для обработки изображений)

## 📋 Требования и установка
1.  Установите локальный сервер (XAMPP/OpenServer).
2.  Скопируйте папку проекта в директорию `htdocs`.
3.  Убедитесь, что в настройках Apache включен `AllowOverride All` для корректной работы `.htaccess`.
4.  Откройте проект по адресу: `http://localhost/my_mvc_site/`

## 📁 Структура проекта
* `application/` — основной код приложения.
    * `core/` — базовые классы MVC.
    * `controllers/` — логика обработки запросов.
    * `models/` — работа с данными.
    * `views/` — интерфейс пользователя.
* `index.php` — единая точка входа.
* `.htaccess` — конфигурация путей.
